pub fn priority_rank(&self) -> i32
[src]

Get the priority rank when they are displayed. A lower number implys a higher priority. The value is intended to use in comparation context and may be changed later, so don't rely on a specific number.

The order of operators are as follows: (From higher to lower)

  • Scalars: Integer, Sym, Approx, Undefined.
  • Pow.
  • Neg.
  • Product, Ratio.
  • Sum.

Examples

use symrs::sym::{Expr, Symbol};

let x = Symbol::new("x");
assert!(Expr::integer(1).priority_rank() < (1i32 + x).priority_rank());
